Essa tag -p- significa criar paragrafo

Para criar um espaçamento entre linhas usar a tag br entre <>

Para criar uma quebra de linha no meio do texto, basa usar a tag br entre <>

Para criar emojis dentro do texto, busca o codigo na emojipedia colocando &#x e depois o codigo, finalizando com ;

Para fazer upload de uma imagem, basta digitar img e escolher a imagem

Para criar o simbolo da pagina, basta digitar link e escolher a opção favicon

Usa-se hx para classificar os niveis de titulos, este valor varia entre h1 até h6
